# 打开说明

## 你现在可以看什么

- `index.html`：企业风格的一页式产品说明书，适合直接打开预览或转 PDF。
- `需求说明.md`：完整产品说明和实施建议，适合继续拆任务。
- `采购清单.md`：淘宝搜索关键词和采购顺序。
- `产品建议.md`：我补充的产品取舍建议，例如尺寸、静音开关、离线模式、权限策略。
- `assets/`：硬件配置图、架构图、接线逻辑图，均为自绘 SVG 示意图。

## 推荐打开方式

直接双击：

```text
需求/index.html
```

如果浏览器因为本地文件限制导致图片不显示，可以在当前项目目录启动一个本地服务：

```bash
python3 -m http.server 8080
```

然后访问：

```text
http://localhost:8080/需求/
```
